**15 좁은 의미의 데이터베아스를 정의하시오.**

In a narrow sense, a database is a collection of data organized for a specific purpose, representing a physical or logical repository of stored data.

**16 데이터베아스의 주요 특성 4가지를 설명하시오.**

The main characteristics of a database include:

1. Data Integration: Management of data from multiple sources in a unified manner.
2. Minimized Data Redundancy: Reduction of duplicate data.
3. Data Integrity: Maintenance of data accuracy and consistency.
4. Data Security: Control and protection of access to data.
